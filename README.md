# Logi-Track 🚚

Logi-Track is a logistics management and tracking system designed for monitoring temperature-sensitive shipments and improving operational transparency in logistics processes.

This project was developed for the **SE 305 / SE 321 – Software Specification and Design** course.

## Project Scope

In this project, the system requirements and architecture of the Logi-Track platform were analyzed and modeled.

The following software engineering activities were completed:

- Identification of **functional and non-functional requirements**
- Development of **Activity Diagrams** for system workflows
- Creation of a **Context Model** to define system boundaries
- Design of **Data Flow Diagrams (DFD)**
- Modeling of system interactions with a **Use Case Diagram**
- Definition of system structure using a **Class Diagram**
- Database structure design with an **Entity–Relationship (ER) Diagram**
- Behavioral modeling using **Sequence Diagrams**
- Interaction modeling with **Communication Diagrams**

These models describe how the system manages shipment tracking, driver operations, notifications, invoicing, and customer monitoring.

## System Components

- Admin Panel
- Driver Mobile Application
- Customer Portal
- IoT Sensor Integration
- Central Database System
